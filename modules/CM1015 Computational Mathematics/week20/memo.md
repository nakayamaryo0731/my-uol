## 1. 代表値（Measures of Central Tendency）

| 用語 | 英語 | 定義 |
|------|------|------|
| 平均値 | Mean | 合計 ÷ 個数 |
| 中央値 | Median | ソートして真ん中 |
| 最頻値 | Mode | 最も頻繁に出現 |

### 中央値の求め方
```
奇数個: 真ん中の値
偶数個: 真ん中2つの平均
```

### いつ何を使う？
```
通常のデータ → Mean
歪んだ分布・外れ値あり → Median
カテゴリデータ → Mode
```

---

## 2. 散布度（Measures of Spread）

### 範囲（Range）
```
Range = 最大値 - 最小値
```

### 分散（Variance）
```
Variance = Σ(x - μ)² / n
```

### 標準偏差（Standard Deviation）
```
SD = √Variance
```

---

## 3. 正規分布（Normal Distribution）

### 68-95-99.7 ルール
```
μ ± 1σ : 約68%（片側34%）
μ ± 2σ : 約95%
μ ± 3σ : 約99.7%
```

### 図解
```
      |--34%--|--34%--|
      |       |       |
------+-------+-------+------
     μ-σ     μ      μ+σ
```

### 例：μ=14, σ=1 で13時間以上の確率
```
13 = μ - 1σ
P(X ≥ 13) = 34% + 50% = 84% = 0.84
```

---

## 4. 計算例

### 分散と標準偏差
```
データ: 2, 4, 6, 8, 10
平均 = 6
差の2乗: 16, 4, 0, 4, 16
分散 = 40/5 = 8
標準偏差 = √8 ≈ 2.83
```

### サイコロの分散
```
目: 1〜6、平均 = 3.5
分散 = 17.5/6 ≈ 2.917
標準偏差 ≈ 1.708
```

## 公式まとめ

| 公式 | 用途 |
|------|------|
| Mean = Σx / n | 平均 |
| Range = Max - Min | 範囲 |
| Variance = Σ(x-μ)² / n | 分散 |
| SD = √Variance | 標準偏差 |
| μ±1σ = 68% | 正規分布 |
| P(A or B) = P(A)+P(B)-P(A∩B) | 加法定理 |
