# Part 1: 確率（Probability）

## 1. 確率の基本

```
確率は 0 から 1 の間の数値

0 = 不可能（impossible）
1 = 確実（certain）
```

## 2. 確率の計算方法

```
確率 = 成功する結果の数 / 全ての結果の数
```

---

## 3. 排反事象（Mutually Exclusive）

**定義：** 同時に起こりえない事象

```
P(A or B) = P(A) + P(B)
P(A and B) = 0
```

---

## 4. 独立事象（Independent Events）

**定義：** 一方の結果が他方に影響しない

```
P(A and B) = P(A) × P(B)
```

---

## 5. 余事象（Complement）

```
P(Aでない) = 1 - P(A)
```

**例：** 少なくとも1つ6が出る（2つのサイコロ）
```
P = 1 - (5/6 × 5/6) = 1 - 25/36 = 11/36
```

---

## 6. 2つのサイコロの合計

| 合計 | 通り数 | 確率 |
|------|--------|------|
| 7 | 6（最多） | 1/6 |
| 9 | 4 | 1/9 |

# Part 2: 順列と組み合わせ

## 1. 順列 vs 組み合わせ

| | 順列（Permutation） | 組み合わせ（Combination） |
|---|---|---|
| 順序 | **重要** | 重要でない |
| 公式 | P(n,r) = n!/(n-r)! | C(n,r) = n!/(r!(n-r)!) |

---

## 2. 公式まとめ

| 状況 | 公式 |
|------|------|
| 全部並べる | n! |
| n個からr個選んで並べる | P(n,r) = n!/(n-r)! |
| n個からr個選ぶ | C(n,r) = n!/(r!(n-r)!) |
| 繰り返し可能 | n^r |

---

## 3. Replacement（復元）

- **With replacement:** 選んだ物を戻す → n^r
- **Without replacement:** 戻さない → P(n,r) または C(n,r)

## 重要公式

| 公式 | 用途 |
|------|------|
| P(A or B) = P(A) + P(B) | 排反事象 |
| P(A and B) = P(A) × P(B) | 独立事象 |
| P(not A) = 1 - P(A) | 余事象 |
| n! | n個全て並べる |
| P(n,r) = n!/(n-r)! | 選んで並べる |
| C(n,r) = n!/(r!(n-r)!) | 選ぶだけ |
| n^r | 繰り返し可能 |
