# Q7 独自性ステートメント（日本語訳）

本プロジェクトはゼロから構築した2つの新しいチャートタイプを導入した：GitHubのContribution Graphに着想を得たTraining Heatmap（ネストループと色マッピングにより日別距離をカレンダーグリッドに表示）と、三角関数を使って5つの正規化指標をスパイダーチャートにプロットするMarathon Readiness Radar。また、ピアソンの相関係数と最小二乗法による線形回帰も実装した — これらはモジュールのカリキュラムを超えた統計手法であり、数学的公式から自主的に調査・実装した。これらの機能は真のギャップに対応している：Garmin Connectは個別のメトリクスを表示するが、心拍数とペースのトレンドを相関させたり、全体的なマラソン準備度を単一の統合ビューで評価したりすることはできない。

（93語）
